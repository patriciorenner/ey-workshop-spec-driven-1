# Workshop 1 — Fundamentos de Spec-Driven Development con GitHub Copilot

Material del workshop de EY del **22 de abril de 2026**.

Este repositorio contiene únicamente:

- La **presentación** del workshop: [`workshop-materials/EY-SDD-Workshop-1.pptx`](workshop-materials/EY-SDD-Workshop-1.pptx).
- Los **assets** que los participantes copian o consumen durante los hands-on.

No hay código de aplicación, toolchain ni ejercicios completos: el trabajo real se hace en el repo de cada participante, tomando los artefactos de aquí como punto de partida.

## Assets del workshop

Viven en [`workshop-materials/`](workshop-materials/). Ver [`workshop-materials/README.md`](workshop-materials/README.md) para el mapeo completo hands-on → asset → destino en el repo del participante.

## Requisitos para participantes

- VS Code actualizado.
- Acceso a GitHub Copilot (plan individual, Business o Enterprise).
- Git configurado y un repo propio donde pegar los artefactos.

## Idioma

Material en **español**. Identificadores de código, nombres de archivo y frontmatter en inglés.
